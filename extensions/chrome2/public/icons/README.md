# DatResume Chrome Extension Icons

This directory contains the extension icons. You should replace these placeholder files with your actual icons:

- icon-16.png (16x16 pixels)
- icon-32.png (32x32 pixels)  
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

The icons should be in PNG format and represent your DatResume brand.
